2.  Describe the purpose of a DIV tag.
A DIV tag is a container element used to discern certain elements of a webpage. For example, one div can be used to hold all of a webpage's text content and format it accordingly.
It can also be a subset of that example and there can be a DIV element for something like body text or the masthead.
It's purpose is to define and contain a specific element of code.
